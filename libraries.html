Google Maps API Libraries

Libraries extend the functionality of the Google Maps APIs by adding new features, implementing common design patterns, or making some tasks a little easier. The libraries listed on this page are all open source, and have been written by the Google Developer community. If you've worked on a library for the Maps API, we invite you to add your project to the list.

Google Maps JavaScript API

Library Summary Maintainer
canvaslayer A library for adding a <canvas> layer to a Google Maps JavaScript API v3 map, and a small API for efficiently visualizing data using the 2d canvas API or WebGL. Brendan Kenny
googleearth A library integrating the Google Earth API with the Google Maps JavaScript API v3, rendering most map overlays appropriately in the 3D plugin. Josh Livni
infobubble A InfoBubble is a customizable CSS info window. Luke Mahe
maplabel A library that adds well-styled, customizable text to a particular location in a Google Maps JavaScript API v3 map. Chris Broadfoot,
Luke Mahe
markerclusterer A Google Maps JavaScript API v3 library to create and manage per-zoom-level clusters for large numbers of markers. Luke Mahe
richmarker A library for using arbitrary DOM elements as map markers in the Google Maps JavaScript API v3. Luke Mahe
storelocator This library enables developers to easily build store locator-type applications using the Google Maps JavaScript API v3. Chris Broadfoot
